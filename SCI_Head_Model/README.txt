
------------------------------------------------------------------------------------------
Diffusion Tensor Images (DTI):

This folder contains all necessary files related to DTI data in this project.
---
The following files are the eigenvalues for the DTI tensor field:
  DTI_L1.nrrd
  DTI_L2.nrrd
---
The following files are the orthogonal eigenvectors for the DTI tensor field:
  DTI_V1.nrrd
  DTI_V2.nrrd
  DTI_V3.nrrd
---
DTI_tensors.nrrd: DTI tensor field.
---
MRI_to_DTI_SCIRun_transform.mat: Transformation matrix for MRI to DTI registration.
---
Networks: Contains all SCIRun networks used for DTI data in this project.
